# 🏠 Rent Calculator (Python)

A simple Python project that calculates how much each person should pay when sharing the expenses of a rented house or flat.

This project was created to practice Python fundamentals such as:

* Variables
* User input
* Arithmetic operations
* Output formatting

## 🚀 Features

* Calculates the total electricity bill.
* Adds rent, food expenses, and electricity costs.
* Splits the total amount equally among all people.
* Displays a clear bill summary.

## 🛠️ Technologies Used

* Python 3

## 📂 Project Structure

```text
rent_calculator.py
README.md
```

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python rent_calculator.py
```

## 💻 Example

### Input

```text
Enter your monthly rent (£): 1200
Enter the food expense (£): 300
Enter the electricity units used: 250
Enter the charge per electricity unit (£): 1
Enter the number of people sharing: 3
```

### Output

```text
----- Bill Summary -----
Rent: £1200
Food: £300
Electricity Bill: £250
Total Expense: £1750
Each Person Pays: £583.33
```
